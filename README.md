### 🚀 About Me

I am a passionate **Generative AI Engineer** specializing in building scalable LLM applications, RAG systems, voice agents, and multi-agent workflows. I focus on bridging the gap between humans and computers using state-of-the-art Generative AI models, vector stores, and advanced audio processing pipelines.

*   💼 **Currently Open to Opportunities:** Seeking roles as a Generative AI / LLM / Agentic Systems Engineer.
*   🤖 **AI Focus:** Multi-agent orchestration, semantic search pipelines, conversational STT/TTS voice interfaces, and workflow automations.
*   🛠️ **Newly Developed Featured Projects:**
    *   **[AI Farmer Assistant](https://github.com/Sunil-Gajula25/Farmer_Bot):** An AI-powered smart agriculture platform that diagnoses crop leaf diseases using MobileNetV2 + DeepSeek AI, provides organic/chemical treatments, and guides farmers. Check out the [walkthrough demo](https://github.com/Sunil-Gajula25/Farmer_Bot/blob/master/walkthrough.webp).
    *   **[AURA Transcriber](https://github.com/Sunil-Gajula25/multilanguage-transcriber):** A premium full-stack tool for 16-language video/audio transcription and phonetic romanization using Whisper, Librosa audio cleanup, and FastAPI.
    *   **[Qwen RAG PDF Chatbot](https://github.com/Sunil-Gajula25/rag-pdf-chatbot):** An end-to-end local document inspector with semantic retrieval using sentence-transformers, FAISS vector indexing, and local Qwen2.5-3B LLM.
